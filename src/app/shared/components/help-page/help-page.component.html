This is a small help to guide you through how to create a card.
There are multiple steps to add a card:
  • First we have the 'Title'. A simple text field to add the title you would like.
  • Second we have the 'Photo File'. Simply add a png.
  • Third we have the 'Genres'. You can add the different genres seperated by commas.
  • Fourth we have the 'Release Date'. Simply add the release date you desire.
  • Fifth we have the 'Description'. Add the description you would like on the card.
